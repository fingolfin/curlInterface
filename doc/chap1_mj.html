1.1-1 DownloadURL
‣ DownloadURL( [URL] )( function )

Returns: A record describing the result. This record will always contain the option 'success', which will be true or false. If 'success' is true, then 'result' will contain the downloaded file. If 'success' is false, then 'error' will contain a human-readable string describing the error.

Download a URL from the internet. Accepts a web address as a string, which can start with either of http:// or https://

1.1-2 URLVerification
‣ URLVerification( [Bool] )( function )

Control https verification. Disabling checking (by passing false) disables checking of https addresses. This reduces security, but can be required for badly configured servers or clients.
